# Claude Tools

A collection of Claude Code skills, tools, and plugins I've built — free and open source.

Each folder is a standalone tool. Clone the whole repo or grab just the one you need.

---

## Tools

| Tool | What it does |
|---|---|
| [linkedin-apollo-research](./linkedin-apollo-research/) | Automates building structured contact profiles for employees at a target company using LinkedIn + Apollo.io |

---

## How to use a skill

Each tool has its own `README.md` with setup instructions. The general pattern:

1. Open the tool's folder
2. Follow the `README.md` to set up any required accounts or extensions
3. Copy the `SKILL.md` contents into a Claude Cowork session
4. Fill in the variables and run
